# pikmi
заменяет текст на пикми-текст по всей системе (совершенно бесполезная шмяга)

### Сборка

```bash
g++ -std=c++17 -o pikmi.exe pikmi.cpp -luser32 -lole32 -loleaut32 -mwindows
```

### Выключение

Если вдруг не смог найти процесс в диспейчере задач то можно закрыть через консоль:
```bash
taskkill /f /im pikmi.exe
```
